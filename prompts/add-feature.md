# Prompt: Add Feature

Use when implementing a new feature from a spec or description.

```
Read AGENTS.md.
Read specs/<relevant-spec>.md if one exists.
Load memory/ documents relevant to the affected area.
Load patterns/ for any applicable implementation recipe.
Read the affected source files.
Read existing tests for the affected area.

Implement the feature described below.
Keep changes small and testable.
Add one test for any non-trivial logic.
Update memory/ or patterns/ if you discover something worth preserving.

Feature: <description>
```
