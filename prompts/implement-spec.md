# Prompt: Implement Spec

Use when a spec exists in specs/ and needs to be built out.

```
Read AGENTS.md.
Read specs/<spec-file>.md in full.
Load all memory/ documents — this is a larger task.
Load relevant patterns/.
Read existing source and tests.

Produce a short implementation plan (≤10 bullet points).
Wait for approval before writing code.

Then implement the spec in small commits:
- One logical change per commit.
- Tests alongside implementation.
- Update ARCHITECTURE.md if the structure changes.
- Extract any discovered knowledge into memory/.

Spec: specs/<spec-file>.md
```
